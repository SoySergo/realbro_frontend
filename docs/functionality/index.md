# Функциональность проекта

## Компоненты объектов недвижимости

### Контакты и связь
- [Contacts](features/property/Contacts.md) - Модалка контактов владельца/агента с проверкой авторизации и лимитов

### Действия пользователя
- [Actions](features/property/Actions.md) - Лайки, заметки, шаринг, жалобы

## UI Компоненты

### Тосты
- `SimpleToastProvider` - Провайдер для простых уведомлений (success, error, warning, info)
- `GlobalToastProvider` - Провайдер для уведомлений о новых объектах

## API

### Контакты
- `getContactAccess()` - Получение контактов с проверкой авторизации и лимитов
- `getCachedContacts()` - Проверка кэша контактов

## Локализация

Новые секции в `messages/{locale}.json`:
- `contact` - Переводы для модалки контактов
- `actions` - Переводы для действий с объектами
