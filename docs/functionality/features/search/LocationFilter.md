# LocationFilter

**Что делает**: Фильтр локации с выбором режима поиска недвижимости по географическим критериям.

**Ключевые особенности**:
- **Четыре режима**: 
  - Полигоны (выбор сохраненных областей)
  - Рисование (создание новой области на карте)
  - Изохрон (время до точки: пешком/велосипед/машина)
  - Радиус (круговая область от точки)
- **UI стиль**: Dropdown с иконками, консистентный с другими фильтрами
- **Интеграция**: При выборе режима триггерит `setLocationMode()` в сторе, что открывает панель деталей (второй ряд фильтров)
- **Стейт**: `useFilterStore()` - `activeLocationMode` хранит выбранный режим

**Файл**: `src/components/features/search/filters/LocationFilter.tsx`

**Связанные типы**:
- `LocationFilterMode`: `'polygon' | 'draw' | 'isochrone' | 'radius'`
- `LocationFilter`: полная структура фильтра локации

**Локализации**:
- `locationPolygons` - Полигоны
- `locationDraw` - Нарисовать
- `locationTimeFrom` - Время до точки
- `locationRadius` - Радиус от точки
