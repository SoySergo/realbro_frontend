# MobileSidebar

**Что делает**: Адаптивный сайдбар для мобильных устройств с верхним меню для управления поисковыми запросами и нижней навигацией.

## Ключевые особенности

### Компоненты
- **QueriesSelect**: Селектор сохранённых поисковых запросов (открывается на весь экран)
- **BottomNavigation**: Нижнее навигационное меню с иконками
- **Кнопки фильтров и сортировки**: Заглушки для будущего функционала

### Поведение
- Верхнее меню показывается только на странице поиска (`/search`)
- При клике на селектор открывается полноэкранный оверлей со списком запросов
- Кнопка "Создать запрос" зафиксирована внизу оверлея (над нижней навигацией)
- Переключатели темы и языка удалены (будут в настройках)

### Локализация
Все UI тексты через `useTranslations('sidebar')`:
- `selectSearch` - заголовок селектора
- `newSearch` - кнопка создания запроса
- `filters` - aria-label кнопки фильтров
- `sort` - aria-label кнопки сортировки
- `close` - aria-label кнопки закрытия
