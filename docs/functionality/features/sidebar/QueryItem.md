# QueryItem

**Что делает**: Переиспользуемый компонент для отображения элемента поискового запроса в компактном или полном виде.

## Ключевые особенности

### Пропсы
- `query: SearchQuery` - объект поискового запроса
- `isActive: boolean` - является ли запрос активным
- `isHovered?: boolean` - наведён ли курсор (для desktop)
- `canDelete: boolean` - можно ли удалить запрос
- `variant: 'compact' | 'full'` - вариант отображения
- `onSelect: () => void` - обработчик выбора запроса
- `onDelete?: () => void` - обработчик удаления запроса

### Варианты отображения

**compact**: Для свёрнутого desktop сайдбара
- Только иконка MapPin
- Tooltip с полной информацией при ховере

**full**: Для развёрнутого сайдбара и мобильной версии
- Иконка + название + статистика + фильтры
- Кнопка удаления (если можно удалить)

### Вложенные компоненты
- `QueryStats` - отображение статистики (количество результатов, новые)
- `QueryFilters` - отображение фильтров в виде badges (локация, спальни, цена)

### Локализация
- `sidebar.bedrooms` - для отображения "bed" / "спален" / "chambres"
