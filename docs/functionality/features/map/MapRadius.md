# MapRadius - Компонент радиуса для карты

**Что делает**: Позволяет пользователю выбрать точку на карте (кликом или через поиск) и нарисовать круг заданного радиуса вокруг этой точки.

## Ключевые особенности

### Компоненты
- **MapRadius** - главный компонент с логикой и интеграцией карты
- **RadiusControls** - UI для выбора радиуса (слайдер)
- **IsochroneAddressSearch** - переиспользуемый компонент поиска адреса с автокомплитом

### Радиусы
- Шаги: 1, 3, 5, 10, 15, 20 км
- Построение круга с помощью библиотеки Turf.js
- Цвет: синий (#3B82F6)

### Технологии
- **Turf.js** - построение геометрических фигур (круг)
- **Mapbox Geocoding API** - поиск адресов и мест
- **Mapbox GL** - отрисовка на карте

### Локализация
- Все тексты UI через `messages/{locale}.json`
- Ключи в секции `radius`
- Поддержка: en, ru, fr

## Использование

```tsx
import { MapRadius } from '@/components/features/map/MapRadius';

<MapRadius map={mapInstance} className="w-96" />
```

## Файлы
- `/src/components/features/map/MapRadius.tsx` - главный компонент
- `/src/components/features/map/RadiusControls.tsx` - UI управления радиусом
- `/src/components/features/map/IsochroneAddressSearch.tsx` - поиск адреса (переиспользуется)

## Тестовая страница
`/test/radius` - демо-страница с полной функциональностью

## Отличия от изохрона
- Не использует внешний API - круг строится локально через Turf.js
- Проще и быстрее - нет сетевых запросов
- Фиксированный радиус в километрах (не зависит от времени/транспорта)
