## 12. Переключатель типа поиска: Недвижимость / Агентства

**Приоритет:** Высокий

### 12.1 Кнопка переключения

- [x] Создать/найти компонент переключателя типа поиска
- [x] Поведение: переключает фильтры и тип отображаемых объектов (недвижимость ↔ агентства)

### 12.2 Десктоп

- [x] Кнопка находится **перед** маркерами (все объекты, нравятся и т.д.)

### 12.3 Мобилка — в фильтрах

- [x] В полноэкранных фильтрах — кнопка переключения типа
- [x] В сайдбаре выбранных (недвижимость/агентства) — при клике показывает select со сменой категории

### 12.4 Привязка к фильтрам

- [x] При смене типа — меняются доступные фильтры
- [x] При смене типа — меняются объекты в листинге/на карте

**Файлы:** `features/search-filters/`, `features/category-filter/`, `widgets/search-filters-bar/`, `screens/search-page/`
**Чекер:** Кнопка переключает между недвижимостью и агентствами. Фильтры меняются. Объекты в листинге соответствуют выбранному типу. На мобилке — в фильтрах и через select.
